Data Saturdays

Following the announcement of the ending of the PASS organisation https://www.pass.org/PASS-Blog/the-future-of-pass,
we, the community, wish to provide a means for events to continue under the Data Saturdays name.
Please feel free to join the discussion for the re-creating of this provision at https://github.com/sqlcollaborative/DataSaturdays

We believe, by sharing these resources and enabling community input, we can continue to help people to provide the same level of events the SQL Saturdays provided. This site and the repo has been set up by Gianluca Sartori, Rob Sewell, Steve Jones, and Johan Ludvig Brattås but is hosted in the SQL Collaborative organisation and is for the community.
Additional contributors presently include Matt Gordon, Andy Mallon, Elizabeth Noble, Monica Rathbun, and Randolph West.
